  • The server address is: http://130.211.153.28:9000/todos
  • API: to get all the todos in the server you send a GET request to http://130.211.153.28:9000/todos
  • API: to add a Todo in the server you send a POST request to http://130.211.153.28:9000/todos with the Todo as a json payload
  • API: to update a Todo in the server you send a PUT request to http://130.211.153.28:9000/todos/HERE ADD THE TODDO _ID with the Todo as a json payload
  • API: to delete a Todo in the server you send a DELETE request to http://130.211.153.28:9000/todos/HERE ADD THE TODDO _ID
  • I remind you that you need to add a service to mangae all server communications
